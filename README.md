プログラミングの道標を作るサイトです

プログラミング学習に参考になりそうなチュートリアルを集めています


## Contributing to CODEROAD
#### リポジトリをクローン
```
$ git clone this repository
```

#### bundle install
```
$ cd coderoad-rails
$ bundle install
```

#### DB作成
```
$ rake db:create
$ rake db:migrate
```

#### RailsServer起動
```
bundle exec rails s
```
